Overview

 A simple, easy to use online expense tool helping employees and management to efficiently process expenditures within an intuitive and robust system.

Main Features & Comparison

 Free version of e-Expense includes the following features:
  • Multi-line expense entry
  • Expense Delegation
  • Basic Reporting Tools
 Advanced version of e-Expense includes:
  • All the features from the basic version
  • Multi-step approval processes
  • Advanced Management Reporting

Multi-line Expense Entry

 Multiple expense lines allows for expense grouping, particularly useful for events such as business trips where all the expenses can be filled under one single expense. This also vastly improves productivity compared to creating one single expense items each time.

Controlled Expense Delegation
